About UCSDMap

Moving the Map

There are three ways to scroll and zoom the map:

  • The controls to the left of the map view
    • The arrow buttons slide the map.
    • The circle resets the map to the default view.
    • The plus and minus buttons zoom in and out.
    • The ovals between the plus and minus buttons change the selected zoom level.
  • Clicking and dragging inside the map view
  • Keyboard shortcuts (see below)

Additionally, clicking on one of the location names that appear to the left of the map centers on that location.

Keyboard Shortcuts

Key Function
+ (plus) Zoom in
- (minus) Zoom out
Arrow keys Scroll the map in the given direction

Note: Not all key commands work in all browsers.
